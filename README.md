## 📦 安装

```bash
$ git clone https://github.com/wangz-code/ng-zorro-admin.git ng-zorro-admin
$ cd ng-zorro-admin
$ npm install

```

## 📝 开发环境

```bash
$ ng serve --open
```

## 🎉 打包

```bash
$ ng build --prod
```
